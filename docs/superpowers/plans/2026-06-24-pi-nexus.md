# pi-nexus Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a publishable pi package `pi-nexus` that registers the Nexus Router as an OpenAI-compatible provider so its chat models can drive pi.

**Architecture:** A single async extension factory discovers downloaded chat-capable models from `GET /models?show_all=true`, probes each to confirm it serves text, then registers the survivors under a `nexus` provider using pi's built-in `openai-completions` API with `thinkingFormat:"qwen"` (exact match for Nexus's `enable_thinking`/`reasoning_content`). Pure filtering/mapping logic is extracted for unit testing; live HTTP is isolated to the factory.

**Tech Stack:** TypeScript, pi Extension API (`@earendil-works/pi-coding-agent`), Node `fetch`, `node:test` for unit tests.

## Global Constraints

- Base URL is `https://api.nexus-projects.ai/api/v1` (note `/api/v1`).
- Auth token format `nxr_...`, sent as `Authorization: Bearer <token>` (no `authHeader` flag — OpenAI SDK does it).
- Provider name is `"nexus"`. Env var is `NEXUS_API_KEY`.
- Cost is `{input:0,output:0,cacheRead:0,cacheWrite:0}` (Nexus meters server-side).
- All registered models: `reasoning:true`, `input:["text"]`, `compat.thinkingFormat:"qwen"`, `contextWindow:131072`, `maxTokens:16384`.
- Only `downloaded===true` models with `modality==="text"` OR `recipe==="collection.omni"` are candidates, and each must pass a live probe before registration.
- No custom `streamSimple` — built-in openai-completions handles everything.
- Package mirrors `pi-tokenrouter` structure: `package.json` with `pi.extensions:["./extensions"]`, `type:"module"`, `peerDependencies`.

```typescript
import type { ProviderModelConfig } from "@earendil-works/pi-coding-agent";

export interface NexusModel {
  id: string;
  modality?: string;
  labels?: string[];
  recipe?: string | null;
  downloaded?: boolean;
}

export function isChatCapable(m: NexusModel): boolean {
  if (m.recipe === "collection.omni") return true;
  return m.modality === "text";
}

export function toProviderModel(m: NexusModel): ProviderModelConfig {
  return {
    id: m.id,
    name: m.id,
    reasoning: true,
    input: ["text"],
    api: "openai-completions",
    contextWindow: 131072,
    maxTokens: 16384,
    cost: { input: 0, output: 0, cacheRead: 0, cacheWrite: 0 },
    compat: { thinkingFormat: "qwen" },
  };
}

export function mapNexusModels(models: NexusModel[]): ProviderModelConfig[] {
  return models
    .filter((m) => m.downloaded === true && isChatCapable(m))
    .map(toProviderModel);
}
```

### Task 3: The extension factory

**Files:**
- Create: `extensions/nexus.ts`

**Interfaces:**
- Consumes: `mapNexusModels` from `../lib/mapping.ts`, `process.env.NEXUS_API_KEY`, `fetch`.
- Produces: default-exported async extension that registers the `nexus` provider.

- [ ] **Step 1: Implement the async factory**

Responsibilities: resolve key, fetch `/models?show_all=true`, map with `mapNexusModels`, **probe each candidate** (skip errors), register provider with provider-level `X-Nexus-Agent` header.

```typescript
import type { ExtensionAPI, ProviderModelConfig } from "@earendil-works/pi-coding-agent";
import { mapNexusModels, type NexusModel } from "../lib/mapping.ts";

const BASE_URL = "https://api.nexus-projects.ai/api/v1";
const PROVIDER_NAME = "nexus";
const ENV_API_KEY = "NEXUS_API_KEY";

export default async function (pi: ExtensionAPI) {
  const apiKey = process.env[ENV_API_KEY];
  if (!apiKey) {
    console.warn(`[pi-nexus] ${ENV_API_KEY} not set. Skipping.\n  export ${ENV_API_KEY}=nxr_...`);
    return;
  }

  let models: NexusModel[];
  try {
    const res = await fetch(`${BASE_URL}/models?show_all=true`, {
      headers: { Authorization: `Bearer ${apiKey}` },
    });
    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    const data = (await res.json()) as { data?: NexusModel[] } | NexusModel[];
    models = Array.isArray(data) ? data : data.data ?? [];
  } catch (err) {
    console.warn(`[pi-nexus] model discovery failed: ${err instanceof Error ? err.message : err}`);
    return;
  }

  const candidates = mapNexusModels(models);
  if (candidates.length === 0) {
    console.warn("[pi-nexus] no downloaded chat-capable models found.");
    return;
  }

  // Probe each candidate; skip any that don't serve text.
  const verified: ProviderModelConfig[] = [];
  for (const m of candidates) {
    const ok = await probeModel(m.id, apiKey);
    if (ok) verified.push(m);
    else console.warn(`[pi-nexus] skipping ${m.id} (probe failed / no backend)`);
  }

  if (verified.length === 0) {
    console.warn("[pi-nexus] no models passed probe; not registering.");
    return;
  }

  const agent = (process.env.NEXUS_AGENT || "pi").slice(0, 128);
  pi.registerProvider(PROVIDER_NAME, {
    name: "Nexus Router",
    baseUrl: BASE_URL,
    apiKey: `$${ENV_API_KEY}`,
    api: "openai-completions",
    headers: { "X-Nexus-Agent": agent },
    models: verified,
  });
  console.log(`[pi-nexus] registered ${verified.length} models under "${PROVIDER_NAME}".`);
}

async function probeModel(id: string, apiKey: string): Promise<boolean> {
  try {
    const res = await fetch(`${BASE_URL}/chat/completions`, {
      method: "POST",
      headers: { Authorization: `Bearer ${apiKey}`, "Content-Type": "application/json", "X-Nexus-Agent": "pi-probe" },
      body: JSON.stringify({
        model: id,
        messages: [{ role: "user", content: "Reply with exactly: pong" }],
        stream: false,
        max_completion_tokens: 64,
        enable_thinking: false,
      }),
    });
    return res.ok;
  } catch {
    return false;
  }
}
```
